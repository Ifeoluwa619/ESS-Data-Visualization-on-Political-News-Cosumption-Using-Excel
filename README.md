# ESS Data Visualization on Political News Consumption Using Excel

This repository presents data visualizations that explore patterns of political news consumption among residents of the **United Kingdom** and **Italy**, using data from the **European Social Survey (ESS)**.

## Project Overview

Understanding how people consume political news provides insight into political engagement, media exposure, and cross-national differences.  
This project uses **Microsoft Excel** to analyze and visualize political news consumption patterns based on ESS data, demonstrating how accessible tools can be used for effective data storytelling.

The project focuses on:
- Visual exploration of political news consumption
- Cross-country comparison between the UK and Italy
- Clear and interpretable Excel-based visualizations

## Data Source

The data used in this project were obtained from the **European Social Survey (ESS)**, a reputable cross-national survey that collects data on attitudes, beliefs, and behaviors across European countries.

## Tools and Methods

- **Microsoft Excel**
  - Data cleaning and preparation
  - Pivot tables
  - Charts and visualizations (bar charts, comparative visuals)
- Descriptive analysis to identify patterns and differences in news consumption
